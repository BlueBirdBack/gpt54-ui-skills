# UI review checklist

## Quick pass
- Does it still look like the library component family?
- Is the visual role obvious?
- Is width appropriate?
- Is spacing calm?
- Is text wrapping where it should?
- Is hierarchy clear?
- Is contrast sufficient but not theatrical?
- Is placement intentional?
- Did we solve it with the minimum change?

## Overlay / map UI
- Is it floating cleanly instead of becoming a strip/blob?
- Does it fight the map/header visually?
- Is the contrast enough without going muddy-black?
- Is the element readable at a glance?
- Does it avoid covering more than necessary?
